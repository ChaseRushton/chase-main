## ğŸš€ Quick Start

### Prerequisites

- Node.js >= 14.0.0
- npm >= 6.14.0

### Installation

1. Clone the repository
```bash
git clone https://github.com/ChaseRushton/chase-main.git
```

2. Install dependencies
```bash
cd chase-main
npm install
```

3. Start development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000)
